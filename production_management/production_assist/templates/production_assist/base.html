Production Management Assist

Application purpose is to show essential skills in programming.

Used technologies: Django, Python, jQuery, HTML5, CSS (Bootstrap), PostgreSQL

Application contains basic tools to manage flow of information within manufacturing company.
This includes information about cooperating companies with personnel contact data, offers and retails that cooperating companies ordered previously.

Custom made pagination NavBar

For custom pagination I have created new View based on generic ListView.
In PaginatedListView function get_context_data() get extended by new variable page_range that is used later on if page is paginated.
When page is paginated additional template is loaded which determine formatting of pagination NavBar.

Custom functions in Model.Manager

Standard Model.Manager for Company model and Person Model is upgraded with custom function search for easy filtering data on Company/Person List View.
